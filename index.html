Negócios AI & BI

Flexibilidade

Fuzzy permite tratar variáveis que não são apenas verdadeiras ou falsas, mas que podem ter diversos graus de verdade!


Isso é útil para problemas complexos ou que há muitas incertezas.









Realismo

Lógica Fuzzy considera as diferenças subjetivas e as incertezas envolvidas!

Muito útil quando os resultados podem ser influenciados por diferentes opiniões ou perspectivas.

Facilidade no uso

Pode ser implementada com auxílio de software específico, uma linguagem de programação pré-definida ou mesmo uma planilha de cálculo.

Facilita para avaliar diferentes alternativas.

Adaptação

Sistemas Fuzzy permitem a adaptação conforme necessidades específicas de cada situação.


Útil para avaliar fatores diferentes que variam de um caso para outro.
